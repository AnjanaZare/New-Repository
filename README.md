# MySQL-backed Java App

This project is a Java application that connects to a MySQL database.

## Tech Stack
- Java
- Spring Boot / JDBC / Spring Data
- MySQL

## Setup
1. Configure database credentials in `src/main/resources/application.properties`.
2. Run migrations or create schema if required.
3. Build and run the app.

## Commands
- Maven: `./mvnw spring-boot:run`
- Gradle: `./gradlew bootRun`
